Embark LMS — Wireframes

Low-fidelity wireframes for PRD v5 (offline payments + live classes + recorded content). Two roles: Student and Admin. Yellow annotation boxes describe behavior, states, and rules for each screen — these become the build spec for Antigravity sessions.
